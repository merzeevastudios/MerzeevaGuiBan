# 🛠️ MerzeevaGuiBan — Конфигурации и Языки

Добро пожаловать в официальный репозиторий конфигураций для плагина **MerzeevaGuiBan** от студии **MerzeevaStudios**!

Здесь вы можете найти готовые шаблоны настроек и переводы интерфейса на разные языки.

## 📂 Что внутри / What's inside:
* `config(ru).yml` — Конфигурация плагина c описанием на русском языке.
* `config(en).yml` — Конфигурация плагина с описанием на английском языке.
* `config(es).yml` — Конфигурация плагина с описанием на испанском языке (Spanish).
* `config(de).yml` — Конфигурация плагина с описанием на немецком языке (German).
* `config(fr).yml` — Конфигурация плагина с описанием на французском языке (French).
* `config(pt).yml` — Конфигурация плагина с описанием на португальском языке (Portuguese).
* `config(zh).yml` — Конфигурация плагина с описанием на китайском языке (Chinese).
* Папка `languages/` — Готовые файлы локализации для сообщений чата / Ready translation files:
    * `ru.yml` — Русский (Russian)
    * `en.yml` — Английский (English)
    * `es.yml` — Испанский (Spanish)
    * `de.yml` — Немецкий (German)
    * `fr.yml` — Французский (French)
    * `pt.yml` — Португальский (Portuguese)
    * `zh.yml` — Китайский (Chinese)

## 🚀 Как использовать:
1. Выберите нужный файл конфигурации выше.
2. Скопируйте его содержимое и вставьте в локальный файл `plugins/MerzeevaGuiBan/config.yml` на вашем сервере.
3. Скачайте нужный перевод из папки `languages` и положите его в аналогичную папку на сервере.
4. Выполните команду `/merzeevaban reload` для мгновенного применения настроек!
