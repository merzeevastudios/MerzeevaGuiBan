# 🛠️ MerzeevaGuiBan — Конфигурации и Языки

Добро пожаловать в официальный репозиторий конфигураций для плагина **MerzeevaGuiBan** от студии **MerzeevaStudios**!

Здесь вы можете найти готовые шаблоны настроек и переводы интерфейса на разные языки.

## 📂 Что внутри:
* `config(ru).yml` — Конфигурация плагина с водянкой и описанием на русском языке.
* `config(en).yml` — Конфигурация плагина с водянкой и описанием на английском языке.
* Папка `languages/` — Готовые файлы локализации для сообщений чата (`ru.yml`, `en.yml`).

## 🚀 Как использовать:
1. Выберите нужный файл конфигурации выше.
2. Скопируйте его содержимое и вставьте в локальный файл `plugins/MerzeevaGuiBan/config.yml` на вашем сервере.
3. Скачайте нужный перевод из папки `languages` и положите его в аналогичную папку на сервере.
4. Выполните команду `/merzeevaban reload` для мгновенного применения настроек!
